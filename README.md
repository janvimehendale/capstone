## Autism Prevalence AnalysisThis project provides insight on the increasing prevalence of Autism Spectrum Disorder, which is a neurodivergent condition that affects how an individual learns, communicates, behaves, and processes everything happening around them.The purpose of the study is to understand the trends associated with the increase in the early identification of ASD. How have autism prevalence rates changed over time and what demographic or geographic conditions correlate with the trend. The project uses publicly available data from CDC, healthdata.gov and NIH. The assumption is the available data is accurate with the challenge that there might be many undiagnosed or unreported cases.#### Data Question1. How have autism prevalence rates changed across U.S. states from 2000 to recent years?2. Which demographic or geographical location show the strongest associations with these trends?3. What are the gender differences in autism prevalence?4. How does prevalence differ across age groups?5. What types of services are available to help individual with ASD?#### Data Sourceshttps://catalog.data.gov/dataset/autism-prevalence-studieshttps://healthdata.gov/CDC/autism-prevalence-studies/ggvy-6bjb/about\_datahttps://www.cdc.gov/autism/data-research/autism-data-visualization-tool.htmlhttps://www.cdc.gov/autism/addm-network/index.htmlhttps://www.nimh.nih.gov/health/topics/autism-spectrum-disorders-asd#hts-https://www.vecteezy.com/free-vector/autism#### Contact Information##### Janvi Mehendaleemail 	 - janvimehendale@gmail.comlinkedin - https://www.linkedin.com/in/janvi-mehendale/GitHub	 - https://github.com/janvimehendale
